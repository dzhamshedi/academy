Инструменты тестирования

DevTools (Инструменты разработчика)

Встроенные в браузер инструменты для отладки и анализа веб-приложений. Позволяют:

  • Исследовать HTML/CSS структуру страницы
  • Отслеживать сетевые запросы
  • Анализировать производительность
  • Эмулировать мобильные устройства
  • Отлаживать JavaScript

Документация Chrome DevTools

Postman

Популярный инструмент для тестирования API. Функциональность:

  • Отправка HTTP-запросов
  • Автоматизация тестирования API
  • Создание коллекций запросов
  • Генерация документации
  • Совместная работа над API

Официальный сайт Postman

ApiDog

Комплексная платформа для разработки и тестирования API, объединяющая функционал нескольких инструментов:

  • Дизайн и документирование API в визуальном редакторе
  • Тестирование и отладка запросов
  • Автоматическая генерация тестов из спецификации API
  • Встроенный mock-сервер
  • Совместная работа над API в команде
  • Поддержка CI/CD и автоматизации

ApiDog объединяет функциональность Postman, Swagger Editor, Swagger UI, Stoplight, ReadMe, JMeter и других инструментов в одном приложении.

Официальный сайт ApiDog

Swagger (OpenAPI)

Инструмент для документирования и тестирования REST API:

  • Интерактивная документация API
  • Возможность выполнять запросы прямо из документации
  • Автогенерация клиентских библиотек
  • Стандартизация описания API

Официальный сайт Swagger

Git

Система контроля версий, необходимая для:

  • Отслеживания изменений в коде
  • Работы с разными версиями приложения
  • Совместной работы над проектом
  • Ведения истории изменений

Официальный сайт Git

DBeaver

Универсальный инструмент для работы с базами данных:

  • Подключение к различным типам БД
  • Выполнение SQL-запросов
  • Просмотр и редактирование данных
  • Создание ER-диаграмм

Официальный сайт DBeaver

Charles Proxy

Прокси-сервер для анализа сетевого трафика:

  • Перехват и анализ HTTP/HTTPS запросов
  • Модификация запросов и ответов
  • Эмуляция медленного соединения
  • Отладка мобильных приложений

Официальный сайт Charles

Fiddler

Прокси-сервер для отладки веб-трафика:

  • Перехват и анализ HTTP/HTTPS трафика
  • Мощный скриптовый движок для автоматизации
  • Удобный интерфейс для анализа запросов
  • Поддержка веб-сокетов

Официальный сайт Fiddler

Proxyman

Современный прокси для macOS:

  • Нативное приложение для macOS
  • Поддержка HTTP/HTTPS/SOCKS
  • Встроенный сертификат для iOS
  • Удобный интерфейс и быстрая работа

Официальный сайт Proxyman

Wireshark

Анализатор сетевого трафика:

  • Захват пакетов на низком уровне
  • Поддержка множества протоколов
  • Глубокий анализ сетевых проблем
  • Мощные фильтры для поиска

Официальный сайт Wireshark
